EZID search uses AND logic which means ALL terms entered into a search box must be present in order to show up in the results.
    • Example AND search: "trees fruit" will return any search results with both keywords trees and fruit.
If you would like to use OR logic which means ANY terms entered in the search box can be present in the results, please add an "OR" to the search query.
    • Example OR search: "trees OR fruit" will return any search results with the keywords trees or fruit.
