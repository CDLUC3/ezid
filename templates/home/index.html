EZID (easy-eye-dee) makes it easy to create and manage unique, persistent identifiers.

  • You can create identifiers for any kind of entity â€” physical, digital, abstract, etc. For identifiers of objects on the web, you can use EZID to maintain their current locations so that people who click on the identifiers are correctly forwarded.
  • You can also store citation metadata with identifiers to aid in interpreting and maintaining them. Several citation formats are possible.
  • EZID supports a number of persistent identifier technologies, including ARKs (Archival Resource Keys), DOIs (Digital Object Identifiers), and URNs (Uniform Resource Names).
  • Visit our website for more information about EZID identifiers, features, and benefits.

To operate EZID use the tabs above or the programming interface. You will need a login account to create identifiers, but anyone can view identifier metadata. Please contact us if you are interested in setting up an account.
